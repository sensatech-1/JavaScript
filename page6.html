- Try/Catch Block:

} catch (error) {

}
- Throwing Errors:

function checkAge(age) { if (age < 0) throw new Error("Age can't be negative"); console.log("Age is valid"); } - Finally Block:
try {
// code
} catch (error) {
// handle error
} finally {

Function Declarations and Expressions
What is a Function
A function is a reusable block of code that performs a specific task. Think of it like a recipe: you can follow the same recipe multiple times to make the same dish.

- Function Declaration:
`function greet()
{ console.log("Hello!"); }`
- Real-world analogy:
A restaurant menu - you can order the same dish (function) multiple times.
Example:
function addNumbers() {
let num1 = 5;
let num2 = 3;
console.log(num1 + num2);
}
addNumbers(); // outputs 8
