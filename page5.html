Conditional Statements (if/else)
- If/Else Statements:
let age = 18;
if (age >= 18) {
console.log("You can vote!");
} else {
console.log("You can't vote yet.");
}
- Switch Statements: let day = "Monday";
switch (day) {
case "Monday":
console.log("Start of the week!");
break;
default:
console.log("It's a day.");
}

Key points:
- Use `if/else` for conditions, `switch` for specific values
- Don't forget `break` in `switch` üòÖ



Loops (for, while, do-while)
- For Loop:
for (let i = 0; i < 5; i++)
{
console.log(i); // 0, 1, 2, 3, 4
}
- While Loop: let i = 0;
while (i < 5) {
console.log(i); // 0, 1, 2, 3, 4
i++;
}
- Do-While Loop:
let i = 0;
do {
console.log(i); // 0, 1, 2, 3, 4
i++;
} while (i < 5);
Key points:
- `for`: best for counting/iterations
- `while`: when you don't know iterations upfront
- `do-while`: runs at least once üòä
